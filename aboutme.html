My Projects

The Floor is Lava : This was one of the first projects I completed with one of my peers. "The Floor is Lava" is a side scroller game made using Scratch.
Drawing shapes: This project was completed in Python. It used user input to recieve number of sides, number of petals, color of outline, and color of shape.
Obamicon: As part of his 2008 Presidential campaign, Barack Obama released a poster of a certain style. Using Python, I worked with my peers to recreate this filter. Move your mouse over the photo to see before and after shots using this filter applied to a photo of the CCTV Headquarters in Beijing, China.
Django Blog : This blog was created using the Django framework in the languages HTML, Javascript, and CSS. We updated code in Python, then pushed it to Python Anywhere before pulling it to our website.
Dictionary Attack: We programmed a game that asked for password input and would respond with whether or not this password was secure. Passwords that were words in a dictionary, or used simple number-letter replacements (e.g. A=4) were all considered too simple
Robotics: We used this arduino to program and wire a robot that would turn around if it impacted a surface. It was built to play a game called "Escape the Box", in which the robot has to manuver around a box until it can locate the opening.
